# 指数比价分析 (Index Compare)

分析 A 股与港股主要指数的比价关系——**A 股以沪深300为锚，港股以恒生指数为锚**——计算历史分位和均值回归，生成 Plotly 交互式 HTML 报告。

## 锚定体系

| 市场 | 比价对（分子 / 分母） | 交易逻辑 |
|---|---|---|
| **A 股** | 中证500 / 沪深300 | 比价分位：分子便宜买分子，分母便宜买分母 |
| | 中证1000 / 沪深300 | 同上 |
| | 创业板 / 沪深300 | 同上 |
| | 上证50 / 创业板 | 同上（风格跷跷板） |
| | 300价值 / 300成长 | 同上（价值/成长跷跷板） |
| | 科创50 / 上证50 | 同上（科技/超大盘跷跷板） |
| **港股** | 恒生科技 / 恒生指数 | 同上 |

> **所有比价对的分子和分母都是可交易指数，比价只是一个相对贵贱的跷跷板信号——哪边便宜就配置哪边。**
>
> **特殊之处**：沪深300 和恒生指数除了参与比价对之外，各自还有独立的**股权溢价指数（ERP）**框架，用于判断其自身绝对估值的历史分位高低。ERP 由外部项目（Equity Risk Premium）和报告内置的恒生 ERP 模块分别提供。

## 环境要求

- Python 依赖：`tushare pandas numpy plotly scipy`
- 必须设置 `TUSHARE_TOKEN` 环境变量或在 skill 目录下创建 `.env` 文件

安装依赖：
```bash
pip install tushare pandas numpy plotly scipy
```

## 使用方式

### 完整分析（默认）

```bash
python scripts/main.py
```

自动执行：清理临时文件 -> 检查环境 -> **智能增量更新**（自动检测数据是否需要更新）-> 计算比价和分位 -> 生成智能分析 -> 输出 HTML 报告。

**增量更新说明**：
- 首次运行获取全部历史数据（2007年至今，约19年）
- 后续运行自动检测本地与远程数据，只获取新增交易日数据
- 如需强制完整更新，使用 `--force` 参数

### 强制完整更新

```bash
python scripts/main.py --force    # 强制重新获取全部历史数据
```

### 快速查询（使用已有数据）

```bash
python scripts/main.py --query           # 查询所有指数
python scripts/main.py --query ZZ500     # 只看中证500
python scripts/main.py --query ZZ1000    # 只看中证1000
python scripts/main.py --query ZZA500    # 只看中证A500
```

快速查询不重新获取数据，使用上次分析的缓存结果。

### 手动清理临时文件

```bash
python scripts/cleanup.py --force        # 强制清理所有临时文件
python scripts/cleanup.py --max 10       # 自定义阈值
```

主程序启动时会自动清理（`tmpclaude-*` 文件数量 >= 20 时触发）。

## 分析指标

详细的分析规则见 `analysis-rules.md`。核心指标概要：

- **历史分位数**：基于全部历史数据（2007年至今，约19年）。< 20% 极度低估（超配），20-40% 相对低估，40-60% 中性（标配），60-80% 相对高估，> 80% 极度高估（低配）
- **均值回归**：偏离30日均线。> +5% 超买，< -5% 超卖，±5% 内正常
- **趋势判断**：比较当前比价与 5/10/20 日前数据，判断上升/下降/震荡
- **配置建议**：综合趋势、分位、偏离度三个维度生成超配/标配/低配建议

## HTML 报告内容

报告包含：指标卡片（比价/分位/偏离度）、价格走势图（近2500交易日，约10年）、三个比价走势图（含分位数标注和红绿虚线标注区间极值）、智能分析结论和配置建议。深色主题，金融终端风格。
